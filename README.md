# Compiler-C-C-in-asm-16-bit
C compiler in 16-bit assembly, with syntax similar to C++
for the software license read the appropriate file "LICENSE.md"

# English
In development

To compile, use the syntax "compiler *name of the file to compile* -o *name of the output file*"

The file "base.h" is used to define the functions for the code editor so that it does not give various errors due to the lack of definitions

The file "syntax.md" contains the code syntax. For understanding, find the C code of which you want the explanation in the file and then read the explanation in the next line with the translation in asm.
