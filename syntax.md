# #define ORG
Sintassi "#define ORG *valore*"
Il comando asm "org" serve per impostare al compilatore del asm il punto di inzio del file rispetto alla ram in modo tale che i puntatori delle variabili puntino agli indirizzi di memoria assoluti corretti(il valore deve essere quello del indirizo di memoria in cui viene caricato il file)
